Thank you for your purchase of the Potions Pack!

Potions Pack contains: 
1. EPS vector files with 99 potions (9 forms of 11 colors) + EPS vector file with 2 cauldrons (side and top view). Fully editable and resizable. SVG vector files are also included.
2. Preview images of the pack (format: PNG)
3. Readme.txt file

- How to open svg-file?
This art was created in Inkscape. So you can open svg-file using Inkscape - a free and open source vector graphics editor (www.inkscape.org). Also, svg-format is supported by Adobe Illustrator, CorelDRAW and other applications. You can open SVG-file and save in any needed size.
